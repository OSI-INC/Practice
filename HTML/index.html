Introduction

[29-DEC-23] Hi there! Welcome to the Practice/HTML demonstration directory. Open the "index.html" file in a text editor. That way you will be able to see the raw HTML code. Open it again in a web browser. That way you will also be able to see what the page looks like when it is rendered. When you edit and save "index.html", reload the file in the web browser to see your changes.

Now, this should be in italics and this should be in bold. This sentence will also be in the same paragraph, even though there are line breaks in the HTML file. When a browser renders the HTML, it treats line breaks like spaces, and it treats multiple spaces like one space. The paragraph does not end until you see an "end p tag".

So this text appears in a new paragraph. Here are some special characters: × ÷ −.

Figures

Here we have a picture. The picture is centered along with its caption by use of a "center" tag. The centering ends later with an "end center" tag. We use an "img" tag to specify an image in the same directory as our HTML file. The image is "A3036C.jpg". We specify that the picture should be drawn with width "800", which is usually supposed to mean "800 pixels", but you never can tell, really.


Figure: An Implantable Optical Stimulator.

The image does not have to be local, it could be an image that resides on a website. The image below is on the OSI website. We specify the image with a web link.


Figure: A Coin Cell Battery with Tabs Welded.

The figure caption we add after a "br" tag. The "br" tag generates a line break. The "small" tage makes small text.

Video

Starting with HTML5, we can use a video tag to embed a video in our page.


Video: Example Video. Press play to start.

The video above will play in its embedded window, and appears in the HTML just like an image. The "video" tag is similar to an "img" element, except that we must end it with an "end video" tag.

Stylesheets

You will see a "cascading style sheets" reference at the top of this HTML document. The style sheet is "screen.css". It defines how paragraphs and headings are supposed to look, how big the text should be, and what font to use. It specifies justification and paragraph indenting.
